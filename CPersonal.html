Protocolos de internet (TCP/IP) -1982-

Se establece el protocolo TCP/IP, un conjunto de protocolos de red en los que se basa Internet y que permiten la transmisión de datos entre redes de computadoras y surge el término Internet, por «Inter-Networking».

NFS net -1985-

Se crea la Network File System (sistema de archivos de red), o NFS, es un protocolo de nivel de aplicación. Es utilizado para sistemas de archivos distribuido en un entorno de red de computadoras de área local.
