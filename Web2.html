Web 2.0

Wikipedia -2001-

El 15 de enero de 2001 se crea Wikipedia, la enciclopedia libre, como proyecto de edición abierta que pretendía agilizar y flexibilizar el desarrollo de la enciclopedia virtual.

Facebook -2004-

Mark Zuckerberg funda la plataforma facebook, volviéndose la principal red social en el mundo, iniciando el auge de las redes sociales.

Gmail -2004-

Google anuncia su servicio de correo electrónico: Gmail.

AJAX -2005-

Se crea la tecnología Ajax, una técnica de desarrollo web que, nos permite intercambiar información entre el servidor y el cliente (un navegador web) de forma asíncrona.

Youtube -2005-

Se funda YouTube, una plataforma para reproducir vídeos, subir contenido a un canal propio e interactuar con el resto de usuarios.

Amazon EC2 -2006-

Se crea Amazon Elastic Compute Cloud (Amazon EC2), un servicio web que proporciona capacidad de computo de tamaño ajustable en la nube. Está diseñado para facilitar a los desarrolladores recursos de computación escalables basados en web.
